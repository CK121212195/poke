## 🌟 Overview
**Pokémon Gacha** is a web-based, fully free Gacha simulator where you can encounter infinite unseen Pokémon! Turn the wheel of destiny, and a random Pokémon card will be dispensed from the capsule. 

This application uses real Pokémon data and card images fetched from the official [PokéAPI](https://pokeapi.co/) and [Pokémon TCG API](https://pokemontcg.io/).

## ✨ Features
- **Turn the Gacha**: Experience a fun animation and sound effects when turning the crank.
- **Pokémon Cards**: Displays actual Pokémon Trading Card Game (TCG) artwork. (Falls back to official artwork if a card is unavailable).
- **Detailed Pokédex Info**: View base stats, abilities, height, weight, category, and flavor text.
- **Authentic Cries & BGM**: Plays the official Pokémon cries upon reveal, alongside immersive Opening and Battle background music.
- **Region Selection**: Choose to pull Pokémon from specific regions (Kanto, Johto, Hoenn, etc.) or from all regions (1-1025).
- **Bilingual Support**: Toggle seamlessly between English and Japanese interfaces.
- **Share on X (Twitter)**: Easily copy the card image and share your pulls with your friends on X.

## 🛠 Tech Stack
- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS
- **APIs**: [PokéAPI](https://pokeapi.co/), [Pokémon TCG API](https://pokemontcg.io/)
- **Assets**: FontAwesome (Icons), Canvas Confetti (Effects)

## 🚀 Getting Started
Since this is a client-side only web application, no complex installation is required.

1. Clone this repository:
   ```bash
   git clone https://github.com/pythonddd/pokemon.git
   ```
2. Make sure you have `op.mp3` (Opening BGM) and `battle.mp3` (Battle BGM) in the same directory.
3. Open `index.html` in your favorite web browser.

*(Alternatively, you can host it easily using GitHub Pages!)*

## 🌟 概要
**ポケモン ガシャポン (Pokémon Gacha)** は、カプセルの中に広がる無限の出会いを楽しめる、完全無料のWebガチャシミュレーターです！運命のハンドルを回して、ランダムなポケモンカードをゲットしましょう。

本アプリケーションは、[PokéAPI](https://pokeapi.co/) および [Pokémon TCG API](https://pokemontcg.io/) から本物のポケモンデータとカード画像をリアルタイムで取得して表示しています。

## ✨ 主な機能
- **ガチャ演出**: ハンドルをタップすると、アニメーションと効果音とともにカプセルが排出されます。
- **ポケモンカード表示**: 実際のポケモンカードの画像を表示します（カードが見つからない場合は公式イラストを表示）。
- **詳細な図鑑データ**: 種族値、特性、高さ、重さ、分類、図鑑説明文（フレーバーテキスト）を確認できます。
- **本物の鳴き声＆BGM**: ポケモン登場時に公式の鳴き声が再生され、オープニングや戦闘BGMも楽しめます。
- **地方の選択**: カントーからパルデアまで、特定の地方に絞ってガチャを回すことができます（全地方も選択可能）。
- **言語切り替え**: 英語と日本語のUIをワンタップで切り替え可能です。
- **X（旧Twitter）シェア**: ゲットしたポケモンの画像をクリップボードにコピーし、そのままXへ投稿できます。

## 🛠 使用技術
- **フロントエンド**: HTML5, Vanilla JavaScript, Tailwind CSS
- **外部API**: [PokéAPI](https://pokeapi.co/), [Pokémon TCG API](https://pokemontcg.io/)
- **その他**: FontAwesome（アイコン）, Canvas Confetti（紙吹雪エフェクト）

## 🚀 使い方（ローカル実行）
本プロジェクトはフロントエンドのみで動作するため、複雑な環境構築は不要です。

1. リポジトリをクローンします。
   ```bash
   git clone https://github.com/pythonddd/pokemon.git
   ```
2. 同じディレクトリ内にBGMファイルとして `op.mp3`（オープニング用）と `battle.mp3`（戦闘用）を配置してください。
3. `index.html` をブラウザで開くだけで遊べます。

*（GitHub Pagesなどを使用して、簡単にWeb上に公開することも可能です）*
